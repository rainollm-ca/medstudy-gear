# Rabiaa Printful Connection Steps

When the WooCommerce site is live:

1. Go to `https://medstudy.rainomotion.com/wp-admin`.
2. Log in with the admin credentials I provide privately/through 1Password.
3. In WordPress sidebar: `Printful` → `Connect`.
4. Approve WooCommerce access.
5. Printful will redirect you to your Printful account.
6. Choose/connect the store.
7. In Printful: Billing → Billing methods → add primary billing method for this store.
8. Back in WordPress: WooCommerce → Settings → Shipping → Printful Shipping → Enable.
9. In Printful: Add products using `/projects/medstudy-gear/designs/png/` files.
10. Submit products to store and confirm they appear under WooCommerce → Products.

You still personally complete:
- Printful login/OAuth
- Printful billing
- Stripe/PayPal KYC/payment setup
- Any tax/legal business details
